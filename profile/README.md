# 🌿 Checkerly: Pure Trade, Healthy Root

> **"불투명한 중고시장을 정화하는 해독제, 체커리(Checkerly)입니다."**

**Checkerly**는 팀 **Root**가 가꾸는 AI 기반 안심 중고거래 생태계입니다. 
치커리가 땅속 독소를 정화하듯, 우리는 AI 판별 기술을 통해 중고 시장의 사기와 기만(Fake)을 걸러내고 가장 신선하고 정직한 거래만을 연결합니다.

---

## 🌱 Our Ecosystem (Core Value)

- **Root (관리자: 팀 루트):** 서비스의 근간을 지탱하는 강력한 백엔드 엔진입니다. 모든 데이터의 무결성을 검증하고 생태계의 건강함을 유지합니다.
- **Seed (사용자: 신규 가입자):** 체커리의 생태계에 합류한 소중한 씨앗들입니다. 투명한 거래를 통해 신뢰의 꽃을 피웁니다.
- **Detox (검증):** 오염된 이미지와 허위 매물을 AI로 스캔하여 제거합니다. 깨끗한 시장을 만드는 체커리만의 핵심 기술입니다.

---

## 🏗️ System Architecture



Checkerly는 씨앗들이 안전하게 자랄 수 있도록 견고한 인프라를 제공합니다.

- **The Root (Backend):** Spring Boot 3 & PostgreSQL. 모든 비즈니스 로직의 근간이자 AI 검증 엔진의 심장부입니다.
- **The Sprout (Frontend):** React Native. 사용자가 가장 먼저 만나는 싱그러운 인터페이스입니다.
- **AI Scanner:** 실물 사진 여부를 판별하여 사기의 싹을 잘라내는 정밀 분석 레이어입니다.

- **[📂 Checkerly-Frontend](https://github.com/Checkerly/Checkerly-Frontend)**: 씨앗들을 위한 앱 서비스 및 루트 전용 백오피스

## ✨ 핵심 차별화 기능 (Core Features)

1. **실시간 'Fresh Check' 요청:** 채팅 중 즉석에서 실물 사진을 요청하여 이미지 도용 여부를 실시간으로 정화합니다.
2. **AI 트러스트 스코어 (신선도 점수):** 전송된 사진의 실물 확률을 분석하여 해당 매물의 '신선도'를 수치로 제공합니다.
3. **루트 전용 백오피스 (Root Admin):** 모든 판별 로그를 모니터링하고, 오염된 데이터(사기 유저)를 즉각 제재하여 생태계를 보호합니다.
